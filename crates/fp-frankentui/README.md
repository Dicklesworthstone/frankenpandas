# fp-frankentui

FrankenTUI terminal interface for interactive fp-frame / frankenpandas
workflows.

Part of the [frankenpandas](https://github.com/Dicklesworthstone/frankenpandas)
workspace.

## Status

Early development. Surface still evolving — not yet part of the
public stable API of frankenpandas (the umbrella crate). Direct
dependency only for experimental TUI demos until the surface
stabilizes.
